## About
This is a backend application for booking vehicle service appointments. This application is built on NestJs framework and uses Mongo Db as database.

## Installation

```bash
$ npm install
```

## Running the app

```bash
$ nest start
```

## Test

```bash
# unit tests
$ npm test
```
